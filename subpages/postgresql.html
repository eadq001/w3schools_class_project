PostgreSQL Tutorial

PostgreSQL is an advanced relational system.

PostgreSQL supports both relational (SQL) and
non-relational (JSON) queries.

PostgreSQL is free and open-source.

Learning by Doing

In this tutorial you get a step by step guide on how to install and create a PostgreSQL database. You will
learn how to create a project where you can create, read, update, and delete data.

You will learn how to query, filter, and sort data from the database.
